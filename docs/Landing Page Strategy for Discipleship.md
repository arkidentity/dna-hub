# Design Guidelines

## Visual Identity
- **Color Palette:** Warm colors (cream/gold/earth tones, NOT dark/cinematic)
- **Aesthetic:** Trusted mentor, approachable, professional but warm
- **Photography:** Natural lighting, real people if available (not stock photos)

## Typography
- **Headings:** Clean sans-serif (Inter, Helvetica, or similar)
- **Body:** Readable serif or sans-serif with good line spacing
- **Hierarchy:** Clear visual distinction between headline/subhead/body

## Layout
- **Single column, mobile-first**
- **Ample white space** (don't cram)
- **Above the fold:** Hero + primary CTA visible immediately
- **Sections clearly defined** with spacing
- **CTAs stand out** with color contrast

## Performance
- **Load time:** Under 2 seconds
- **Mobile responsive:** 50%+ will view on phone
- **Forms:** Simple, fast to fill, clear labels
- **Images:** Optimized for web (compressed but quality)

## User Experience
- **One clear path:** Read â†’ Download Manual â†’ Take Assessment
- **No friction:** Minimal form fields, immediate delivery
- **Trust signals:** Privacy note, what to expect, no pressure language
- **Accessibility:** Readable font sizes, good contrast, semantic HTML
